# **App Name**: Resonance Tracker

## Core Features:

- Time Conversion: Display the current time in both the user's local time zone and Blue Protocol's time zone (UTC-2). Allow users to manually adjust the displayed time zone.
- World Boss Timers: Display a timer showing when the world bosses will spawn. Show countdown for bosses with :00, :30 and every 2 hour spawn timers.
- Interactive Map: Display an interactive map with key locations for timed events (world bosses, etc.). Allow users to zoom and pan around the map.
- Kanamia Event Display: Display the Kanamia event from Dec 5 - Dec 29, every Fri, Sat, and Sun at 10:44, 11:09, 13:56, 14:20, 17:08, 17:32, 20:20, 20:44, 23:32 and 24:56 (UTC-2), there is a Spirit Dance event lasting for 10 minutes.

## Style Guidelines:

- Primary color: Deep blue (#2E3A87) to reflect the Blue Protocol universe.
- Background color: Dark gray (#2A2A2A) to give a modern, immersive feel. Suitable for a dark color scheme.
- Accent color: Electric purple (#BE29EC) for highlighting timers, important events and calls to action. A contrasting hue analogous to the primary, but brighter.
- Body and headline font: 'Inter' for a clean, modern, and readable design. Inter is a sans-serif.
- Use clear and recognizable icons to represent different world boss spawn locations and event types.
- Design a clean, responsive layout that adapts to different screen sizes, making it easy to view information on any device.
- Incorporate smooth transitions and subtle animations to provide feedback when timers update or events trigger.
